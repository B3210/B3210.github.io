memcpy（） 函数

C 库函数 void *memcpy（void *dest， const void *src， size_t n） ==将 n 个字符从内存区域 src 复制到内存区域 dest。==

声明

以下是 memcpy（） 函数的声明。

参数

  • dest − 这是指向要复制内容的目标数组的指针，类型转换为 void* 类型的指针。
  • src − 这是指向要复制的数据源的指针，类型转换为 void* 类型的指针。
  • n − 这是要复制的字节数。

返回值

此函数返回指向目标的指针，即 str1。
